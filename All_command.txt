🚫 6. .gitignore FILE BANANA (venv ignore karne ke liye)

❗ Ye zaroori hai — venv GitHub pe push nahi hona chahiye.




🧠 7. GIT REPO INITIALIZE KARNA

git init



🔗 8. GITHUB REPO SE LINK KARNA

git remote add origin https://github.com/username/repo-name.git
Replace username and repo-name with your GitHub details.



✅ 9. GIT PUSH KARNA

git add .
git commit -m "Initial commit with venv and requirements"
git branch -M main
git push -u origin main
